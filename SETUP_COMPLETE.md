## 🚀 Quick Start - Upload to GitHub

### Option 1: Using Command Line (Recommended)

```bash
# 1. Create repository on GitHub (https://github.com/new)
# 2. Copy the repository URL from GitHub

# 3. Add GitHub as remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/SmartHome-Finder.git

# 4. Rename branch to main
git branch -M main

# 5. Push to GitHub
git push -u origin main
```

### Option 2: Using GitHub Desktop (Easier)

1. Download GitHub Desktop: https://desktop.github.com
2. Click "File" → "Add Local Repository"
3. Select your project folder
4. Click "Publish repository"
5. Fill in repository name and description
6. Click "Publish Repository"

## 📞 Need Help?

### Common Issues:

**"fatal: remote origin already exists"**
```bash
git remote remove origin
git remote add origin https://github.com/YOUR_USERNAME/SmartHome-Finder.git
```

**"Permission denied"**
- Use HTTPS instead of SSH
- Or generate SSH key and add to GitHub

**"fatal: 'origin' does not appear to be a 'git' repository"**
```bash
git init
git remote add origin https://github.com/YOUR_USERNAME/SmartHome-Finder.git
```
